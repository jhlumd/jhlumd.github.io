SoriCloud ( live | github ) is a SoundCloud clone built with React, Redux, Ruby on Rails, and Postgres.

I focused on demonstrating my proficiency with React and Redux by creating a fluid and responsive interface for audio playback and site navigation.

Under the hood, I designed RESTful APIs and optimized the backend by avoiding N+1 queries to allow for scale.

Museic ( live | github ) is a social platform with a unique UI to record and share short melodies. It showcases my ability on the MERN stack (MongoDB, Express, React, and Node.js).

It features a responsive interface for recording melodies and a beautiful real-time visualization of the melody as it is played to enhance user engagement. User input is saved directly into the database as an array of music notes to eliminate the need for audio file data storage.

Animated Image Histogram
( live | github ) creates an animation of the transition between a digital image and its image histogram.

Using only JavaScript and the canvas API, I replotted each pixel using its lightness value in the HSL color space as its new x coordinate and then stacked up all the pixels with similar lightness values and animated the transition using an interpolation function.
